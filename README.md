# Potato Leaf Disease Detection

## Overview
This project focuses on detecting diseases in potato leaves using deep learning techniques. The goal is to develop an accurate and efficient model to classify different types of potato leaf diseases and assist farmers in early disease identification.

## Features
- Image-based classification of potato leaf diseases
- Deep learning model trained using a dataset of diseased and healthy leaves
- Deployment-ready solution for real-time disease detection

## Dataset
The dataset consists of images of potato leaves categorized into multiple disease classes, including:
- Healthy
- Early Blight
- Late Blight

The images are preprocessed and augmented to improve model performance.

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib
- streamlit
